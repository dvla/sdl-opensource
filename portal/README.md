Portal
========

This project contains a sample portal which shows how you could use the sml-token-service and pdf-generator.

## Requirements

 * Play framework 2.2+
 * Scala Build Tools (SBT)
 * Java JDK 1.7+
 * Maven 3

## Build

To build this project simply execute the following SBT goals:

```bash
  sbt ~run
```
